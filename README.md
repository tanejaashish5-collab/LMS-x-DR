# LMS-x-DR

AI-Powered Learning Management System × Demand Radar Platform

## Overview
An intelligent business opportunity platform that combines:
- **LMS (Learning Management System)**: Structured learning paths for business opportunities
- **DR (Demand Radar)**: Real-time market demand analysis and opportunity scoring

## Features
- 🎯 12 AI-analyzed business opportunities
- 📊 Multi-dimensional scoring system
- 🤖 9 specialist AI agents for guidance
- 📚 Learning paths for each opportunity
- 💼 Implementation roadmaps
- 🔍 Market demand tracking

## Tech Stack
- **Frontend**: Next.js 15, TypeScript, Tailwind CSS
- **Backend**: Supabase (coming soon)
- **AI**: Claude API via specialized agents
- **Deployment**: Vercel
